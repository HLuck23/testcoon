Menyoo / Trainer - so spielst du "img_9742_move" ab
=====================================================

WICHTIG: Diese Animation existiert nur auf dem FiveM-SERVER, auf dem die
Ressource "img_9742_move" laeuft (da wo auch /img_9742_move im Chat geht).
Im normalen GTA-Singleplayer ist sie NICHT vorhanden -> dort spielt sie nicht.

Menyoo einrichten:
  1. Datei oeffnen:  ...\menyooStuff\FavouriteAnims.xml
  2. Diese Zeile als eigene Zeile einfuegen (KEIN "pack:/", KEIN ".ycd"):

       <Anim dict="img_9742_move" name="img_9742_move" />

  3. Im Spiel:  Player Options -> Animations -> Favourites -> img_9742_move

Tipp: Startet sie nicht sofort, einmal /img_9742_move im Chat ausfuehren
(laedt die Animation) und danach ueber Menyoo starten.

Dict-Name = Clip-Name = img_9742_move  (beides identisch, ohne Prefix).
